# July

七月，一个用`python3`中的`Django`框架实现的简单小博客，没事自己拿来玩，欢迎指导！

## Quick Start

### Create a new post

``` bash
$ wget https://github.com/anshengme/July/archive/v0.1.0.tar.gz
$ tar xf v0.1.0.tar.gz
$ cd ./July-0.1.0
$ pip install -r requirements.txt
```

### Run server

``` bash
$ python manage.py runserver
```
